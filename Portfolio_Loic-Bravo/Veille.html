Github Copilot







Un outil indispensable pour les développeurs qui cherchent à gagner en productivité.




Qu'est-ce que Copilot ?

Copilot est un outil développé par Github en collaboration avec OpenAI.
Il s'agit d'une intelligence artificielle permettant de générer du code à partir d'une description.
Pour cela, elle s'appuie sur une base de données de plus de 10 millions de lignes de code open source.
Elle est capable de générer du code dans plus de 12 langages de programmation différents.

Comment fonctionne Copilot ?

Copilot fonctionne grâce à l'IA GPT-3 (Generative Pre-trained Transformer 3).
Elle a été entraînée sur une base de données de plus de 175 milliards de paramètres.
Mais le plus intéressant est que Copilot est capable d'apprendre votre façon de coder.
En effet, plus vous l'utilisez, plus il sera performant.

Les autres fonctionalités de Copilot

Copilot est également capable de traduire du code d'un langage à un autre dans sa version "lab".
Cette version est disponible pour les développeurs qui souhaitent tester les fonctionnalités de Copilot en développement.
Il est également possible de l'utiliser pour générer des commentaires de code ou encore de réaliser des tests unitaires.

L'avenir de Copilot

Copilot X, une version plus performante de Copilot, est en cours de développement et a été annoncés.
Celle-ci sera capable de corriger les bugs et de proposer des solutions en sélectionnant une partie de code.
Elle sera également capable de créer des commits et des commentaires de commit de façon autonome répertoriant les modifications apportées.
Elle sera directement intégrée à votre interface de développement sous la forme d'une fenêtre de chat.
La waitlist de Copilot X est déjà ouverte, nous pouvons donc espérer une sortie cette année.
